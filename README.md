# Трекер технологий

Личный пет-проект для организации и отслеживания прогресса в изучении технологий веб-разработки.

## О проекте

Трекер технологий - это веб-приложение, которое я разработал для систематизации процесса изучения новых технологий и инструментов разработки. Приложение помогает планировать обучение, отслеживать прогресс и сохранять важные заметки по каждой изучаемой технологии.

### Предварительные требования

- Node.js (версия 16 или выше)
- npm или yarn

### Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Rithmer/FaBpractice2.git
cd FaBpractice2
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите приложение в режиме разработки:
```bash
npm start
```

4. Откройте [http://localhost:3000](http://localhost:3000) в браузере
